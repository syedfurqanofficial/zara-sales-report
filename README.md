# ZARA Sales Report – SQL Analysis

This project contains a complete SQL-based analysis of ZARA's sales data.  
The goal of this report is to identify revenue insights, product performance,
and top-selling items using structured SQL queries.

## What’s Included
- SQL Queries
- Sales Dataset (sample)
- Summary Report
- Insights & Documentation

## Tools Used
- MySQL / SQL Server
- Excel (for data review)
